This guide explains how to use NOMADs built-in tabular parser to import and structure data from Excel and CSV files. It covers different parsing modes (by column, row, and nested sections), schema annotations, and how to define paths and quantities for automatic data population. The tutorial also demonstrates how to manage multiple entries, use custom YAML schemas, and link data across sections using referencesenabling efficient data handling for experimental inventories, measurements, and structured archives.

This guide explains how to upload and publish data to the NOMAD Repository using supported file formats. It covers file preparation, upload methods, processing, metadata annotation, access control, and DOI assignment. The guide also includes strategies for handling large datasets, automation via API, and special considerations for sensitive data like VASP POTCAR files. Ideal for users looking to contribute FAIR data efficiently and reproducibly.

This documentation introduces the abstract schema used in NOMAD to define, document, and visualize workflows. A workflow describes how data was generated by linking input and output data through tasks. The guide explains how to structure workflow metadata using YAML, including nested workflows and cross-entry references via TaskReference. It also shows how to extend base workflows with custom fields and how workflows are visualized in the NOMAD GUI using the interactive graph tool. These capabilities support clear data provenance and traceable scientific processes.

This tutorial is dedicated to FAIR data management of materials science data based on the NOMAD platform. We will cover the whole data-life cycle: starting with data on your hard drive, we learn how to prepare, upload, publish data, and reference them with a DOI. Furthermore, we will learn how to explore, download, and use data that were published on NOMAD before. We will perform these steps with NOMAD's graphical user interface and its APIs. This is just one installment in a series of tutorials; other tutorials will cover data analysis, lab notebooks, using NOMAD locally (NOMAD Oasis), workflows, and much more.

This tutorial provides a complete walkthrough for uploading, sharing, and publishing research data in NOMAD using both computational and experimental examples. It introduces key components of the NOMAD platform�including uploads, entries, and datasets�and guides users through organizing files, processing data, assigning metadata, and obtaining DOIs. The tutorial also covers the use of NexusDataConverter for experimental data via the NOMAD GUI. By following this guide, users will gain practical skills to structure and publish FAIR-compliant data within NOMAD.
